# Админ-панель и безопасность

Этот файл хранит хронологию завершенных работ, которые относятся к админ-панели, авторизации, загрузке прайсов, маппингу и ограничениям безопасности.

## Формат записи

```md
## ГГГГ-ММ-ДД - Название завершенного плана

- План: `plans/completed/<имя-файла>.md`
- Сделано: что изменилось в администрировании или безопасности
- Влияние: как это влияет на защищенность и операционные процессы
- Проверка: как была проверена работа
```

## 2026-05-19 - Админ-панель и импорт прайс-листов

- План: [plans/completed/phase-5-admin-import.md](../../plans/completed/phase-5-admin-import.md)
- Сделано: добавлены учетные записи администраторов, bcrypt-хэширование пароля, signed cookie, middleware-защита `/admin/*`, страницы входа и защищенные действия загрузки/маппинга.
- Влияние: операционные функции импорта доступны только администратору, а пароль не хранится и не сравнивается в открытом виде.
- Проверка: проверены редирект неавторизованного пользователя, вход администратора, доступ к защищенным страницам и действия маппинга.

## 2026-05-19 - Ограниченный поиск и черный список

- План: [plans/completed/restricted-search-blacklist.md](../../plans/completed/restricted-search-blacklist.md)
- Сделано: добавлен защищенный раздел `/admin/blacklist` для просмотра товаров и включения/выключения флага `is_social_risk`.
- Влияние: администратор может управлять черным списком через существующую модель `Products`, не меняя схему базы.
- Проверка: пользователь проверил раздел админки, выполнены `npm run lint` и `npm run build`.
