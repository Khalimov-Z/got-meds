# Поиск и бизнес-логика

Этот файл хранит хронологию завершенных работ, которые относятся к поиску, fuzzy search, фильтрам, аналогам, правилам выдачи и бизнес-логике GotMeds.

## Формат записи

```md
## ГГГГ-ММ-ДД - Название завершенного плана

- План: `plans/completed/<имя-файла>.md`
- Сделано: что изменилось в логике
- Влияние: как это влияет на пользовательский сценарий или точность выдачи
- Проверка: как была проверена работа
```

## Записи

## 2026-05-11 - Ядро поискового движка

- План: [plans/completed/phase-2-search-engine.md](../../plans/completed/phase-2-search-engine.md)
- Сделано: реализован `searchProducts(query)` с fuzzy search по товарам и алиасам, порогом схожести, сортировкой по `similarity_score`, лимитом выдачи и фильтрацией `is_social_risk`.

## 2026-05-19 - Ограниченный поиск и черный список

- План: [plans/completed/restricted-search-blacklist.md](../../plans/completed/restricted-search-blacklist.md)
- Сделано: `searchProducts(query)` теперь возвращает отдельный restricted-ответ для лучшего совпадения с товаром `is_social_risk = true` или его alias, не включая такие товары в обычную выдачу.
- Влияние: пользователь видит специальный сценарий ограниченного поиска вместо обычного пустого результата, а карта не открывает выдачу по товарам из черного списка.
- Проверка: локально проверены ответы для `диазепам` и `нурафен`, а также пользовательская проверка restricted-заглушек.

## 2026-05-20 - Аналоги препаратов
