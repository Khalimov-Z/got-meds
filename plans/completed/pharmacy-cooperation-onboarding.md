## Пользовательская проверка (Пошаговый чеклист)

### Проверка задачи 1: Отправка заявки владельцем аптеки
1. Запустите локальный сервер (`npm run dev`).
2. Откройте главную страницу `http://localhost:3000`.
3. ✅ **Ожидаемый результат:** Под разделом «Как работает где.таблетка?» отображается карточка сотрудничества с кнопкой «Стать партнером».
4. Нажмите кнопку и перейдите на страницу `/partner`.
5. Заполните форму тестовыми данными:
   * Тип: *Обновить контактные данные*
   * Название аптеки: *Тестовая Аптека*
   * Адрес: *г. Гудермес, ул. Ленина, д. 15*
   * Контактное лицо: *Магомед*
   * Телефон / WhatsApp: *+7 (928) 111-22-33*
   * Дополнительно: *Хотим изменить WhatsApp для связи.*
6. Нажмите кнопку «Отправить заявку».
7. ✅ **Ожидаемый результат:** Форма успешно отправлена, на экране отображается сообщение об успешной отправке и SLA обработки (24 часа).

### Проверка задачи 2: Обработка заявки в админ-панели
1. Войдите в панель администратора `http://localhost:3000/admin`.
2. Перейдите в раздел **«Заявки от аптек»** (или `/admin/partner-requests`).
3. ✅ **Ожидаемый результат:** В списке отображается созданная заявка от «Тестовая Аптека» со статусом **Новая**.
4. Нажмите кнопку **«Отметить как обработанная»** (или аналогичную смену статуса).
5. ✅ **Ожидаемый результат:** Статус заявки меняется на **Обработана**, в списке статус обновляется без ошибок.
