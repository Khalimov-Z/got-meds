# План разработки: Отказоустойчивость импорта и управление синонимами (Mapping & Pipeline Resilience)

**Статус:** Выполнен (Completed)
**Основание:** [spec/features/admin-panel/admin-panel-spec.md](../../spec/features/admin-panel/admin-panel-spec.md), [spec/features/data-pipelines/data-pipelines.md](../../spec/features/data-pipelines/data-pipelines.md)

## Цель
Повысить стабильность импорта CSV-остатков аптек (data pipeline) за счет внедрения безопасного порога (threshold) и защиты от очистки остатков при некорректном парсинге. Добавить в панель администратора интерфейс управления синонимами (алиасами) для возможности оперативного исправления ошибок ручного сопоставления.

## Пользовательская проверка (Пошаговый чеклист)

### Проверка задачи 1: Защита импорта от пустых файлов
1. Откройте панель администратора, перейдите в раздел **«Загрузка остатков»** (`/admin/inventory-upload`).
2. Выберите активную аптеку TIER_2, у которой уже есть загруженные остатки (например, 20+ товаров).
3. Попробуйте загрузить пустой файл CSV (или файл, где все остатки равны 0).
4. ✅ **Ожидаемый результат:** Система блокирует импорт, выводит сообщение об ошибке: *"Импорт заблокирован: количество распознанных товаров (0) снизилось более чем на 50%... Чтобы принудительно затереть остатки, выберите 'Принудительная загрузка'"*. Текущие остатки аптеки в базе данных **не удаляются**.
5. Установите галочку **«Принудительная загрузка»** и загрузите тот же пустой файл повторно.
6. ✅ **Ожидаемый результат:** Импорт проходит успешно, остатки аптеки очищаются.

### Проверка задачи 2: Интерфейс управления синонимами
1. Загрузите файл остатков аптеки с новыми неразспознанными позициями.
2. Перейдите в раздел **«Сопоставление»** (`/admin/mapping`).
3. Сопоставьте неразспознанную строку (например, *"Аспирин Бэби 100мг"*) с препаратом *"Аспирин"* (или любым другим).
4. Перейдите на вкладку **«Список синонимов»** (или `/admin/aliases`).
5. ✅ **Ожидаемый результат:** В таблице синонимов появилась строка: *"Аспирин Бэби 100мг"* -> *"Аспирин"*. Поиск по слову *"Бэби"* находит эту запись.

### Проверка задачи 3: Удаление ошибочного синонима
1. На странице списка синонимов найдите созданный синоним *"Аспирин Бэби 100мг"*.
2. Нажмите кнопку **«Удалить связь»** (или иконку корзины).
3. ✅ **Ожидаемый результат:** Появляется диалоговое окно подтверждения, запись удаляется из таблицы.
4. Перейдите в раздел **«Загрузка остатков»** и повторно загрузите тот же прайс-лист, где содержится *"Аспирин Бэби 100мг"*.
5. ✅ **Ожидаемый результат:** Строка *"Аспирин Бэби 100мг"* снова появляется в очереди неразмеченных позиций на странице `/admin/mapping`, так как её синоним был удален.
