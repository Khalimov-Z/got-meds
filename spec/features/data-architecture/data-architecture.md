## 1. Основные сущности

### 1.1. Аптеки (Pharmacies)
* `id`: UUID
* `name`: Название
* `address`: Строка
* `coordinates`: Latitude, Longitude (для расчета расстояний)
* `tier`: 1 (Киоск), 2 (Крупная частная) или 'Chain' (Сетевая)
* `phone`: Строка
* `whatsapp`: Строка
* `working_hours`: JSON (график работы по дням недели)
* `is_24_7`: Boolean (флаг для ночного фильтра)

### 1.2. Эталонные Препараты (Drugs) - из Apteka.ru
* `id`: UUID
* `name`: Полное правильное название
* `active_ingredient`: Действующее вещество (ключевое поле для аналогов)
* `form`: Форма выпуска (таблетки, сироп, ампулы)
* `is_prescription`: Boolean (флаг рецептурного препарата)
* `price_estimate`: Примерная цена (для определения "базовости")
* `image_url`: Ссылка на фото упаковки

### 1.3. Наличие (Inventory)
Связующая таблица (Many-to-Many).
* `pharmacy_id`: Ссылка на аптеку
* `drug_id`: Ссылка на препарат
* `status`: Enum (`in_stock`, `likely_in_stock`, `unknown`)
* `updated_at`: Дата и время последнего обновления
* `price`: Опционально (если сеть отдает точную цену)

### 1.4. Алиасы маппинга (Drug_Aliases)
* `original_string`: Исходное название из 1С частной аптеки (например, "Нуроф таб")
* `drug_id`: Ссылка на эталонный препарат

## 2. Связи (ER Model)
* Одна `Pharmacy` имеет много `Inventory`.
* Один `Drug` имеет много `Inventory`.
* Один `Drug` имеет много `Drug_Aliases`.
