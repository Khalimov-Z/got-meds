# Пользовательские истории (User Stories) — Оглавление

> **Статус:** Draft
> **Тип:** Сборник пользовательских сценариев (Behavior Driven Development)

Этот раздел содержит максимально детализированные пользовательские истории, описывающие пошаговое поведение системы (что происходит на каждом шаге, куда нажимает пользователь, и как реагирует интерфейс).

## Структура историй

1. 🔍 **[Поиск и навигация](./01-search-and-discovery.md)** — Сценарии нулевого барьера входа, исправления опечаток и поиска аналогов.
2. 🗺 **[Карта и Аптеки](./02-map-and-pharmacy.md)** — Сценарии работы с картой, цветовой индикацией и переходом в мессенджеры.
3. 🛡 **[Локальная специфика и Безопасность](./03-local-rules-and-safety.md)** — Пятничная молитва, фильтрация малых киосков, блокировка соц. риска.
4. 🚀 **[SEO и Пользовательский опыт](./04-seo-and-ux.md)** — Путь пользователя из Яндекса/Google и премиальные анимации интерфейса.
5. ⚙️ **[Администрирование](./05-admin-panel.md)** — Сценарии работы менеджера по обновлению CSV и маппингу базы.
6. 🎛 **[Дополнительные функции и Аналитика](./06-additional-features.md)** — Режим списка, фильтры "Открыто сейчас", звонки, трекинг дефицита и дисклеймер.
