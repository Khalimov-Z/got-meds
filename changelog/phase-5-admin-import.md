# Changelog: Этап 5 — Админ-панель и импорт прайс-листов

**Дата:** 2026-05-19
**Ветка:** `feature/phase-5-admin-import`
**План:** [phase-5-admin-import.md](../plans/completed/phase-5-admin-import.md)

---

## Backend & Data

### Новое
- **Модель `Admin`** — учетные записи администраторов с bcrypt-хэшем пароля и ролью.
- **Модель `UnmappedString`** — очередь строк из прайс-листов, которые требуют ручного маппинга.
- **Расширение `ProductAlias`** — поддержка ignored alias через `isIgnored` для мусорных строк из прайсов.
- **Миграция `20260516120000_add_admin_import_tables`** — таблицы и поля для админского входа, алиасов и очереди маппинга.
- **Seed-администратор** — тестовая учетная запись из `SEED_ADMIN_EMAIL` / `SEED_ADMIN_PASSWORD` с безопасным хэшированием пароля.

### Логика
- Добавлена серверная авторизация администратора через signed cookie и middleware-защиту `/admin/*`.
- Добавлен Server Action `uploadPharmacyPrice` для загрузки CSV-прайса Tier 2 аптеки.
- Реализован full sync остатков: перед новой загрузкой очищаются старые записи `Inventory` выбранной аптеки.
- Распознанные строки обновляют `Inventory` со статусом `LIKELY_IN_STOCK`.
- Нераспознанные строки попадают в `UnmappedString`.
- Добавлены Server Actions `createAlias` и `ignoreAlias` для ручного связывания и игнорирования строк.
- Добавлена проверка формата файла и обязательных колонок `Название` / `Остаток`.

## Frontend & UI

### Новое
- **Страница `/admin/login`** — форма входа администратора.
- **Страница `/admin`** — стартовая админ-панель со статистикой и переходами.
- **Страница `/admin/inventory-upload`** — выбор Tier 2 аптеки, загрузка CSV/XLS и отчет по импорту.
- **Страница `/admin/mapping`** — очередь нераспознанных строк, поиск эталонного препарата, действия «Связать» и «Игнорировать».
- **Служебная ссылка «Администратор» на главной странице** — компактный вход в защищенную админ-зону.

### Изменено
- Главная страница получила компактную верхнюю навигацию без размещения админского входа в футере.
- Стили поиска расширены под служебную ссылку администратора с сохранением общего визуального направления.
