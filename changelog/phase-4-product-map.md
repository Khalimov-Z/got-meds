# Changelog: Этап 4 — Страница препарата и карта аптек

**Дата:** 2026-05-16
**Ветка:** `feature/phase-4-product-map`
**План:** [phase-4-product-map.md](../plans/completed/phase-4-product-map.md)

## Backend & API

### Новое
- **Server Action `getProductDetails(productId)`** (`src/lib/actions/products.ts`) — получает данные препарата для SSR-страницы продукта, исключая препараты из социального риска.
- **Server Action `getPharmaciesByProduct(productId, lat, lng, isOpenNow)`** (`src/lib/actions/products.ts`) — возвращает активные аптеки с наличием, статусом, графиком, контактами и расстоянием.
- **API-эндпоинт `GET /api/pharmacies`** (`src/app/api/pharmacies/route.ts`) — клиентский контракт для повторной загрузки аптек после геолокации, ручного выбора точки и переключения фильтра.

### Логика
- Добавлена Tier-логика: для базовых нерецептурных препаратов Tier 1 показывается со статусом `unknown` без записи в `Inventory`.
- Добавлен фильтр «Открыто сейчас» на основе `working_hours` и `is_24_7` с часовым поясом Гудермеса.
- Добавлен расчет `distance_meters` по Haversine при наличии координат пользователя или ручной точки.
- Добавлена сортировка аптек по расстоянию, а без координат — по приоритету Tier и названию.

## Frontend & UI

### Новое
- **SEO-страница продукта `/product/[id]`** (`src/app/product/[id]/page.tsx`) с серверными metadata, данными препарата, рецептурным предупреждением и переходом на карту.
- **Страница карты `/map?q={запрос}`** (`src/app/map/page.tsx`) с серверным поиском выбранного препарата и начальной загрузкой аптек.
- **Компонент `MapExperience`** (`src/components/map/map-experience.tsx`) с режимами «Карта» и «Аптеки рядом».
- **Интерактивная карта Яндекс.Карт** с маркерами аптек, цветами статусов, выбором аптеки и fallback-схемой при недоступности скрипта.
- **Карточка аптеки / Bottom Sheet** с адресом, графиком, дистанцией, статусом наличия и действиями «Позвонить», «Написать в WhatsApp», «Проложить маршрут».

### Изменено
- Карточки результатов поиска на главной странице стали ссылками на `/product/[id]` без изменения визуального стиля.
- В режиме «Аптеки рядом» карта и список отображаются вместе; выбор аптеки заменяет список карточкой выбранной аптеки.
- Геолокация не блокирует список аптек: начальная выдача доступна сразу, GPS или ручной выбор точки только уточняют расстояния и сортировку.
- Для отказа GPS добавлено компактное действие ручного выбора местоположения на карте.
- В режиме «Аптеки рядом» прокрутка страницы не блокируется картой до явной активации карты пользователем.
- Предупреждения и фильтр «Открыто сейчас» сделаны компактными согласно обновленной дизайн-системе.
