# Changelog (Журнал изменений)

В этой папке хранится хронологическая история разработки проекта GotMeds. 

Каждый раз, когда завершается работа над очередным планом разработки (перенос из `plans/active/` в `plans/completed/`), здесь создается или дополняется запись.

## Структура записей
Чтобы ничего не упустить, все изменения должны быть строго структурированы по следующим областям проекта GotMeds:

* 🗄 **Data & Database:** Схема Prisma, PostgreSQL, расширение pg_trgm, миграции.
* 🧠 **Backend & Core Logic:** Server Actions, алгоритмы поиска (fuzzy search), API эндпоинты.
* 🎨 **Frontend & UI:** Next.js компоненты, Vanilla CSS (glassmorphism), страницы приложения.
* 🗺 **Integrations & Maps:** Подключение Яндекс/2GIS Карт, логика гео-локации аптек.
* 🔒 **Admin & Security:** Авторизация (NextAuth), парсинг CSV-прайсов, логика блокировок (social risk).
* 📈 **SEO & Analytics:** Оптимизация SSR, мета-теги, логирование нулевых запросов поисковика.
* ⚙️ **DevOps & Infrastructure:** Vercel настройки, CI/CD, конфигурации линтеров и тестов.

Это позволяет любому разработчику быстро прочитать и понять, как именно развивался проект с течением времени, не упустив ни одной детали из архитектуры GotMeds.
